# 项目交接文档 — 孔庆丽产品运营经理作品集

## 项目概述

这是一个基于 reveal.js 的幻灯片式个人作品集网页，用于互联网大厂产品运营经理面试。原始用途是百度内部 P7→P8 晋升答辩，现已改写为外部面试作品集语言。

- **技术栈**：reveal.js 4.6.1 + 纯自定义 CSS（无其他框架）
- **设计方向**：暖色调、优雅简洁、杂志风（参考 Yedda / 西门美月匈的个人网站）
- **幻灯片尺寸**：1200×700，fade 过渡
- **启动方式**：本地起一个 HTTP 服务器，浏览器打开 `index.html`

## 关键设计系统（style.css）

- **主色调**：`--accent: #5c4033`（深棕）
- **背景色**：`--bg: #f9f7f4`（暖米白）
- **卡片背景**：`--white: #ffffff`
- **字体**：Cormorant Garamond（英文标题）+ Noto Serif SC（中文标题）+ Noto Sans SC weight 300（正文）

## 交互功能

1. **Lightbox 图片弹窗**：点击任意卡片内的图片可放大查看，点击遮罩/关闭按钮/ESC 退出
2. **卡片详情弹窗（Modal）**：点击任意 `.detail-card` 会弹出该卡片的完整详情克隆（原卡片在页面中可能被截断，modal 中显示完整内容）
3. **键盘导航**：左右箭头翻页， reveal.js 原生支持
4. **幻灯片翻页箭头**：自定义在页面左右两侧中间位置

## 用户核心偏好（务必遵守）

- **不要编造数据**：用户多次强调"别编"，所有数字必须来自她提供的原始材料
- **不要编造职位**：百度只有"产品运营经理"这一个职级称呼，不要发明"高级"等前缀
- **视觉统一**：所有卡片背景保持白色，不要深色浅色混用
- **保持简洁**：默认不写注释，不添加未请求的功能
- **术语对外解释**：DAU→日活用户、CTR→点击率、UV→独立访问量、PV→页面浏览量、次留→次日留存率、CPM/CPA/ROI/ARPU 等如有出现需解释
